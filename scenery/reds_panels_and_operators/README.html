How to Use / Best Practices:

  1. Choose a panel you would like to use, and place it in an empty area where you can build it (do not yet place it where you want it to go).
    1. Double click the panel to open up entity settings.
    2. Choose desired panel and mount colors (if applicable).
    3. Select if you would like to hide any of the optional components (like the mount, etc.). They will not disappear in the editor, but will disappear in Play Mode.
    4. Select what texture you would like to use for the panel. Again, this will only apply in Play Mode.
  2. Choose an operator you would like to add to the panel and place it near the panel.
    1. Double click the operator to open up entity settings.
    2. (Optional) Give it a name. This may be tedious, but will keep your operators organized for easy editing and will greatly help if you run into any errors in Play Mode.
    3. Change the position settings to exactly match the X,Y,Z coordinates of the panel you placed earlier. This should perfectly align the operator to the panel (except for main panels, which also need the X rotation set to -70 and manual alignment; see this tutorial video for how to make this easy).
    4. Adjust position of the operator on the panel as desired.
    5. Like the panel, choose any customization settings you would like for the operator. Operators also have a "Function" setting. Hover your cursor over any of the settings' options for details on what it does.
    6. If you get an error during play mode, read it carefully; you likely forgot to set the block option if using a Function other than "None". Otherwise, please report the bug if possible.
  3. Repeat Step 2 to fully build out your custom panel composition.
  4. Once completed, select your whole composition and press CTRL+C to copy it.
  5. Use the "Click Paste" tool under the "Scenery" tab to paste it near its final location.
  6. While the whole composition is selected, use the "Rotate" tool, along with the basic movement tools, to finely position your custom panel to its final location.
  7. Repeat as desired to place multiple similar, or unique, panels around your ride, and you're basically done! üòÑ
  • As a final note, it might be good to not delete your original "master" panels until you are completed with your build, just in case you need another copy or you want to modify it. Modifying already moved / rotated panels is a pain!!!

Known Issues:

  • "Auto-alignment" via coordinates doesn't work for Main Panels, because they are at an angle. The best practice, in my opinion, is to align one operator by hand, and then copy it's coordinates to other operators you want to add, and adjust from there. This makes aligning any additional operators much easier.
  • "Trains in Service" Mode for Key Switch operators will display the key animation when it shouldn't the first time the ride is put into "Manual Block" Mode. I'm not really sure how to fix this, because it seems like NL2 is miscalculating the number of lashed trains upon entering "Manual Block" Mode for the first time.
  • A LOT of stuff was copy and pasted from object to object. PLEASE let me know if there are any typos or if settings/options are mislabeled.
